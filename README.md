# 🤖 Hugging Face Chatbot with Gradio

A simple AI chatbot built using **Hugging Face Transformers**, **TinyLlama**, and **Gradio**. This project demonstrates how to create a conversational web interface powered by a Large Language Model (LLM) that can be deployed locally or run directly in Google Colab.

---

## 🚀 Features

* Interactive chat interface using Gradio
* Powered by TinyLlama 1.1B Chat Model
* Maintains conversation history
* GPU acceleration when available
* Easy deployment on Google Colab
* Lightweight and beginner-friendly implementation

## ⚙️ How It Works

### 1. Load Tokenizer

The tokenizer converts user text into tokens that the model can understand.

```python
tokenizer = AutoTokenizer.from_pretrained(MODEL_NAME)
```

### 2. Load Model

The language model is loaded and automatically placed on available hardware.

```python
model = AutoModelForCausalLM.from_pretrained(
    MODEL_NAME,
    torch_dtype=torch.float16 if torch.cuda.is_available() else torch.float32,
    device_map="auto"
)
```

### 3. Build Conversation History

Previous messages are converted into a chat format supported by TinyLlama.

```python
messages = [
    {
        "role": "system",
        "content": "You are a helpful AI assistant."
    }
]
```

Conversation history is appended dynamically during each interaction.

### 4. Create Prompt

The chat template transforms conversation messages into the format expected by the model.

```python
prompt = tokenizer.apply_chat_template(
    messages,
    tokenize=False,
    add_generation_prompt=True
)
```

### 5. Generate Response

The model generates a response based on the conversation context.

```python
outputs = model.generate(
    **inputs,
    max_new_tokens=256,
    temperature=0.7,
    do_sample=True,
    top_p=0.9,
    repetition_penalty=1.1
)
```

### Generation Parameters

| Parameter          | Value | Description                          |
| ------------------ | ----- | ------------------------------------ |
| max_new_tokens     | 256   | Maximum length of generated response |
| temperature        | 0.7   | Controls creativity                  |
| top_p              | 0.9   | Nucleus sampling                     |
| do_sample          | True  | Enables probabilistic generation     |
| repetition_penalty | 1.1   | Reduces repetitive outputs           |

## 🎨 Gradio Interface

The chatbot is wrapped inside a Gradio ChatInterface.

```python
demo = gr.ChatInterface(
    fn=chatbot,
    title="🤖 Hugging Face Chatbot",
    description="TinyLlama + Gradio + Transformers"
)
```

Launch the application:

```python
demo.launch(share=True)
```

The `share=True` argument generates a public URL that can be shared with others.
